
Para o exercício 1 foi utilizado java com JDK14.
 
 O arquivo Desafio.java é a classe principal do programa, onde é declarado o array com os valores a serem computados, 
neste arquivo também temos as chamadas aos métodos que fazem os cálculos e concatenações necessárias e no final temos a saída do resultado via console.
  Já no arquivo GeraString.java temos os métodos propriamente ditos. O primeiro método setLista recebe o array por parâmetro e o salva em uma variável, 
no método concatena temos a concatenação dos valores com as frases e a soma dos valores. Por fim temos o método resultado que retorna a string formada 
no método concatena.

Para o exercício 2 utilizei o MySQL.
  
  No arquivo desafio_kopp está a criação do banco de dados, a seleção do banco a ser utilizado, a criação de duas tabelas: remessa e infracao. após temos
comandos que fazem a inserção dos dados nas respectivas tabelas.
  No final do arquivo temos dois select: o primeiro faz a busca por infrações com velocidade igual ou superior a 20% da velocidade permitida e o segundo
consulta quantas infrações cada remessa teve e quantas são validas, invalidas.
